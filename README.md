# go-todo-cli
A simple command line todo list application written in Go.
# Features
- Add tasks
- List tasks
- Mark tasks as done
- Remove tasks
# Installation
```bash
go install github.com/Dorantes98/go-todo-cli@latest
```
# Usage
```bash
make todo

./todo add "Buy groceries"
./todo list
./todo complete 1
./todo remove 1
```
