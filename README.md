# go-todo-cli
A simple command line todo list application written in Go.
# Features
- Add tasks
- List tasks
- Mark tasks as done
- Remove tasks
# Installation
```bash
go install github.com/Dorantes98/go-todo-cli@latest
```
# Usage
```bash
go-todo-cli add "Buy groceries"
go-todo-cli list
go-todo-cli done 1
go-todo-cli remove 1
```
